## 2. 팀원/ 팀장
```
- 여윤모(팀장)
- 김건희
- 김영은
- 김재훈
- 이상녕
- 정현수
```

#### 2.1 담당 업무
```
- 여윤모 : 저장, 언급, 반응
- 김건희 : 리마인더, 파일
- 김영은 : 계정
- 김재훈 : 명령어, 쓰레드
- 이상녕 : 관계형 테이블
- 정현수 : 워크스페이스, 채널
```
